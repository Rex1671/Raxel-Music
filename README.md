# Raxel-Music

A modern, web-based music player built with React and Vite.

## Features

-   **Visualizer**: Real-time audio visualization using Web Audio API.
-   **Themes**: Multiple color themes (Cyberpunk, Ocean, Sunset, Forest).
-   **Sleep Timer**: Auto-stop playback after a set time.
-   **Lyrics**: Synced lyrics display (if available).
-   **Responsive**: Optimized for mobile and desktop.

## Tech Stack

-   React
-   Vite
-   Framer Motion
-   Lucide React
